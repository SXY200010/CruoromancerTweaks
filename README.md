<b><h1>调整内容</h1></b>

融血师修改：
1级 ·聚能灌注：限制每回合1次，5级时 变为DC+2 ，8级时迅捷动作变为自由动作
    · 恶心灌注：限制每回合1次 ，恶心持续轮数为1+法师等级的一半 ，8级时迅捷动作变为自由动作

5级 ·5级的取消法师额外专长变为延后1级（6级获得）并且固定为增强召唤
·号令灌注：力量和魅力的增强加值改为天赋加值，加值目标的标签从骷髅改为死灵生物标签，15级时召唤的死灵生物获得3倍于法师等级的额外临时生命值

15级 增加亵渎灌注：限制每回合1次， 自由动作，下一个使用的死灵法术CL获得+2加值

20级 完美灌注：灌注不再消耗生命，聚能灌注和亵渎灌注的加值提升至+4

死灵学派能力修改：
1级：·驱散死灵额外对敌对死灵生物造成1D4+1/2法师等级+智力调整值的伤害

·墓地之触：改名为不死之触，以一个标准动作，你可以碰触一个不死生物并替它注入负能量。使其攻击跟豁免检定的+1亵渎加值，以及每HD1点的临时生命值。在攻击和豁免的加值会在你每5个法师等级增加1点。这些加值持续法师等级一半的轮数（至少1轮）。你每天可以使用这个能力的次数为“3+智力调整值”。

8级：生命视野改为亡者视野：改为发出一道30英尺范围的灵光，自身与范围内的不死生物在属性检定、攻击检定、豁免检定和伤害检定上获得+1亵渎加值，范围内的敌对活物在属性检定、攻击检定、豁免检定和伤害检定上受到-1减值，并且自身获得盲视。

亡者血承修改：
改名为亡灵血承
3级：惊恐术改为衰弱射线
15级：虚体形态增加被动：获得不死生物面板
20级：增加对偷袭免疫，并且内置增加获得不死生物面板

吸血裔修改 ：·增加吸血鬼共鸣：施放死灵系法术时施法者等级+1/4人物等级
             ·专属专长：饮血者改为也可以直接从活物身上吸血
